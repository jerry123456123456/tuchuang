# 1 编译说明

# 2 功能区别
这是图床第五节课源码，相比上一节课代码的区别，具体使用beyondcompare工具对比tc-mini5和tc-mini4。
主要增加的功能：
1. 图片分享功能使用redis缓存浏览时要通过urlmd5查询的数据，这样能减少mysql的操作压力
2. 增加共享文件列表的获取
3. 增加共享文件下载帮的排行，使用redis做排行
4. 增加文件的删除处理
